YouTube 사이트에 추가된 UI를 통해 영상에 대해 궁금한 점을 간편하게 질문하세요!
- 영상의 페이지 오른쪽 상단에 질문 입력과 [요청] 버튼 추가
- 영상의 더보기 옵션 목록 하단에 [질문하기] 버튼 추가

AI를 활용해 원하는 정보를 효과적으로 확인하고 영상을 볼지 판단할 수 있어요!
- 질문은 YouTube 영상의 정보(제목, 자막)와 함께 ChatGPT로 전달되어 답변 확인

적절한 질문을 통해 원하는 정보를 명확히 얻으세요!
- 직접 질문을 입력하거나 즐겨찾기, 최근 질문 목록에서 쉽게 선택
- AI가 제안하는 질문 추천 기능 활용 (Gemini API Key 설정 필요, 무료로 Key 받기 가능)

최적의 간편한 방법으로 빠르게 질문해보세요!
- 간단히 요청 버튼을 누르면 가장 자주하는 질문으로 요청
- 'q' 단축키를 눌러 질문창 바로 열기

* YouTube 사이트의 사용 경험을 해치지 않도록, 기존 UI와 자연스럽게 어우러지도록 설계되었습니다.
